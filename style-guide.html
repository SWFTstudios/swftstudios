New Page Checklist

Follow this when adding a page or feature to the SWFT 2025 site.

  1. Link css/normalize.css, css/webflow.css, and css/swftstudios2025.webflow.css
  2. Load Google Fonts via WebFont.load (same families as homepage)
  3. Include the .global-styles embed block for utilities and focus states
  4. Use .floating-navbar + .container-3 for navigation consistency
  5. Wrap main content in <main id="lenis" class="main-wrapper"> if using smooth scroll
  6. End with .section.is--footer + .container.footer structure
  7. Use CSS variables (var(--navy), etc.) — never hardcode brand colors
  8. Match button variant to background (light → .button, dark → .btn-primary)
  9. Respect breakpoints: test at 991px, 767px, 479px
  10. For custom CSS, add to a separate file (e.g. css/swftstudios000.css) — do not edit the Webflow export directly
  11. Protected pages (case studies, detail templates) must not be overwritten by Webflow imports — see instructions.md
